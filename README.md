# forwird
A landing page that collects and pushes emails to a list on MailChimp.

[View live site](https://forwird.herokuapp.com/)

## Getting Started

For a local copy, clone the repo and from the command line: `yarn install && yarn start` (or use npm).

(The UI will come up but there will be no functionality. If you want to run it yourself, create a config.json file in the server/config folder with your own MAILCHIMP_API_KEY)

### Features

* Add emails to a Mailchimp-backed subscriber list.

## Built With

* A very simple [Express](https://expressjs.com/) server for hosting on [Heroku](https://www.heroku.com/)
* [Mailchimp API](http://developer.mailchimp.com/)
